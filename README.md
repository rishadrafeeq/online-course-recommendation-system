# Online Course Recommendation System

## Overview

This project is an Online Course Recommendation System developed using Collaborative Filtering techniques.

The system recommends courses to users based on their previous ratings and interactions.

## Features

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Collaborative Filtering (KNN)
- SVD Recommendation Model
- Streamlit Web Application

## Dataset

online_course_recommendation_v2.csv

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- Surprise
- Streamlit
- Matplotlib
- Seaborn

## Run Locally

```bash
pip install -r requirements.txt
streamlit run online_course_dep.py
```
