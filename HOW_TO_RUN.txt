ONLINE COURSE RECOMMENDATION SYSTEM

Requirements:
- Python 3.11 or above

Installation:

1. Clone the repository

git clone https://github.com/YOUR_USERNAME/online-course-recommendation-system.git

2. Move to project folder

cd online-course-recommendation-system

3. Install dependencies

pip install -r requirements.txt

4. Run the application

streamlit run online_course_dep.py

5. Open browser

http://localhost:8501
